CompletableFuture는 비동기 작업에 대한 결과값을 받을 수 있고 외부에서 작업을 완료시킬 수 있게 도와주는 도구이다.

# Future

Java5에 도입된 Future는 비동기 작업에 대한 결과값을 받을 수 있지만 외부에서 작업을 완료시킬 수 없고, 블로킹 코드를 통해서만 이후의 결과를 처리할 수 있다.
이러한 Future를 기반으로 Java8에서 개선되어 나온 것이 CompletableFuture이다.

# 기능

CompletableFuture로 비동기 작업 실행, 작업 콜백, 작업 조합, 예외 처리 작업을 활용할 수 있다.
